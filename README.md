# Pipe Cleaner Crafts 🎨✨

A full-stack eCommerce web application for showcasing and selling handmade pipe cleaner crafts. Users can browse products, add items to cart, place orders, and request custom craft designs. Admins can manage products, orders, and customers through a dedicated dashboard.

## 🚀 Features

### 👤 User Side
- User Registration & Login (Authentication)
- Browse craft products
- Product details view
- Add to Cart functionality
- Secure Checkout process
- Custom craft order requests
- Responsive UI for all devices

### 🛠️ Admin Side
- Admin Dashboard
- Product Management (Add / Edit / Delete)
- Order Management
- View customer custom requests

## 💻 Technologies Used

### Frontend
- React.js
- HTML5
- CSS3

### Backend
